=== Mises à jour de la création de liens

* Visionneuse de codes : Codebox est désormais intégrée au processus de création de liens.  Vous pouvez afficher et copier le modèle CloudFormation depuis Codebox dans Workload Factory avant de le rediriger vers AWS pour exécuter l'opération.
* Autorisations requises : les autorisations requises pour exécuter la création de lien dans AWS CloudFormation peuvent désormais être affichées et copiées à partir de l'assistant Créer un lien dans Workload Factory.
* Prise en charge de la création manuelle de liens : cette fonctionnalité permet la création autonome dans AWS CloudFormation avec enregistrement manuel du lien ARN. Cette fonctionnalité est utile lorsqu'une équipe de sécurité ou DevOps participe au processus de création de liens.
